*# Observability

Repositorio para formación en observability sobre sistemas GNU/Linux.

## Tecnologías que usaremos
* Rsyslog
* logrotate
* Ansible

## Componentes
### Máquinas virtuales
* **Truman:** Red Hat Enterprise Linux observada
* **Christof:** Colector de información

## Dependencias
* Virtualbox o KVM/Qemu
* Vagrant >= 2.3.4
